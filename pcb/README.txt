soldrive_assembly_back.pdf   UXIB-DN12 component placement drawing, rear side

soldriver.kicad_pro          UXIB-DN12 KiCAD 8 project file
soldriver.kicad_sch          UXIB-DN12 schematic
soldriver.kicad_pcb          UXIB-DN12 PCB layout
footprints.pretty/           Project footprint library
sym/                         Project symbol library
